## Run Locally

Clone the project

```bash
  git clone https://github.com/bishtsaurabh60/Basic_calculator.git
```

Go to the project directory

```bash
  cd Basic_calculator
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
